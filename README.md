# Bot para Slack

## Como iniciar

Crea una app en Slack y configurala

https://api.slack.com/apps/new

Crea un entorno virtual y accede a el
```
python -m venv env
```

Instala los requirements
```
pip install -r requirements.txt
```

Crea un archivo de variables de entorno
.env
